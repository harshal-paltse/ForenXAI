Financial institutions today handle an enormous volume of transactions every second, making manual monitoring and auditing nearly impossible. Although machine learning-based systems are used to detect anomalies, most of these systems operate as black boxes, providing little to no explanation for why a transaction is flagged as suspicious. This lack of transparency reduces trust among auditors and regulators, making compliance and decision-making more difficult. Additionally, existing solutions often struggle with high false positives, delayed detection, and inefficient investigation processes. As a result, there is a growing need for an intelligent system that can not only accurately identify anomalous transactions in real time but also provide clear, interpretable explanations, enabling faster, more reliable, and trustworthy financial forensics.
